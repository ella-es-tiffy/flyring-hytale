## üí° Quick Fix: Resetting Config
If your changes aren't appearing in-game, your JSON syntax might be broken.
1.  **Delete** `config.json` in the mod folder.
2.  **Restart** the server.
3.  The mod will generate a fresh, guaranteed working configuration file for you to edit again.
