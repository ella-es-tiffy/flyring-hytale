# IllegalRings Mod

A Hytale server mod that adds five powerful rings with unique abilities. Completely configurable and designed for server owners who want full control over their economy and gameplay balance.

## Features

### 🔵 Fly Ring
- Grants creative-mode flight
- Auto-revokes when ring is removed
- Fall damage immunity while equipped

### 🔥 Fire Ring
- Complete immunity to fire, lava, magma, and burn damage
- Heat-based damage cancellation via ECS system

### 💧 Water Ring
- Immunity to drowning and suffocation
- Safe underwater exploration

### ❤️ Heal Ring (Bloodsuck)
- Lifesteal mechanic (default 5% for endgame balance)
- Heal on dealing damage
- Configurable percentage via `config.json`

### 🕊️ Peaceful Ring
- Complete NPC immunity - Mobs completely ignore you
- If attacked, mobs take damage but do not retaliate
- Player takes 0 damage from mobs if forced interaction occurs

### ⚒️ Dynamic Recipe & Workbench System
- **Fully Customizable Recipes**: Change ingredients and amounts for every ring without recompiling.
- **Workbench Control**: Specify exactly which crafting station (e.g., `Workbench_Tinkering`) and category each ring belongs to.
- **Hidden Recipes**: Turn off crafting individually for specific rings to use them as exclusive Quest rewards or Loot.

## Configuration

The mod uses a sophisticated configuration system located at `mods/tiffy/config.json`. 

### Auto-Sync & Repair
As of **v0.3.1**, the configuration file automatically "heals" itself. If you delete a ring override or leave it empty, the mod will restore the default template at the next startup. Your existing custom settings are always preserved!

### Configuration Example

```json
{
  "_notice": "Illegal Rings v0.3.1 - This file is automatically generated...",
  "enabled": {
    "flyRing": true,
    "fireRing": true,
    "waterRing": true,
    "healRing": true,
    "peacefulRing": true
  },
  "craftable": {
    "flyRing": true,
    "fireRing": true,
    "waterRing": true,
    "healRing": true,
    "peacefulRing": true
  },
  "gameplay": {
    "lifestealPercent": 0.05
  },
  "recipeOverrides": [
    {
      "targetItemId": "Jewelry_Fly_Ring",
      "ingredients": [
        { "id": "Ingredient_Bar_Iron", "amount": 300 },
        { "id": "Ingredient_Life_Essence", "amount": 100 }
      ],
      "benchRequirements": [
        {
          "id": "Workbench",
          "type": "Crafting",
          "categories": ["Workbench_Tinkering"]
        }
      ]
    }
  ]
}
```

### Configuration Options

- **`enabled.<ringName>`**: Toggle power functionality on/off.
- **`craftable.<ringName>`**: If `false`, the recipe is completely hidden from the crafting UI.
- **`gameplay.lifestealPercent`**: Lifesteal percentage for Heal Ring (0.05 = 5%).
- **`recipeOverrides`**: Detailed control over cost and placement.
  - **`id`**: The block ID of the workbench.
  - **`categories`**: The tab/category ID within the workbench.

## Installation

1. Place `FlyRing-0.3.x.jar` in your Hytale server's `mods/` folder.
2. Start the server - `mods/tiffy/config.json` will be auto-generated with "Insane" endgame defaults.
3. Customize ingredients or workbenches in the config and restart the server.

## Technical Details

- **ECS Damage System**: High-priority damage filtering for reliable immunity.
- **Thread-Safe**: Main-thread synchronization for all component updates and network packets.
- **AssetStore Integration**: Recipes are updated at runtime via the Hytale AssetStore API.
